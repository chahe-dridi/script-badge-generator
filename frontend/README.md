# Frontend (React 18)

This directory contains the modernized client-side web application for generating badges. It allows users to:
- Instantly preview badge text and styles with HTML5 `<canvas>`.
- Set advanced configurations like alignment, borders, text color, shadows, and positioning.
- Export all generated badges directly as a ZIP file without needing server-side uploads.

## Built With
- React 18
- CSS (Flexbox / Responsive Design)
- JSZip
- FileSaver

## Quick Start

1. Go to this directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm start
   ```

The application will be accessible at `http://localhost:3001` or `http://localhost:3000`.
